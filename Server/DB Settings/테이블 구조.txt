테이블 설계

account_info
계정 인증정보 테이블 (1개 계정당 1개)
account_unique_id  unsigned not null fill 0 auto increment pk 기본키
account_authLv int not null fill 0
account_email varchar 100 unique
account_password varchar 50
account_guest_token unique varchar 40 unique
account_oauth_token_google unique varchar 40 unique
account_oauth_token_apple unique varchar 40 unique

player_info
플레이어 테이블 (1개 계정당 1개)
player_unique_id unsigned not null fill 0 auto increment pk 기본키
account_unique_id fk 외래키
player_Lv int default 0
player_exp int default 0
player_stamina int
player_ingame_point int
player_cash_point int
player_nickname varchar 12
player_profile_image_url varchar 100
player_gender varchar 5
player_birthdate varchar 11

character_info
캐릭터 테이블 (1개 계정에 여러개 캐릭이 있을 경우)
character_unique_id unsigned not null fill 0 auto increment pk 기본키
account_unique_id fk 외래키
character_name
character_Lv 기본값 0
character_exp
character_hp
character_atk
character_def
character_spec_stat
